# UtilityPro - 유틸리티 구독 서비스 웹 데모

이 프로젝트는 iLoveIMG 스타일의 UI와 광고/구독 기반의 액세스 제어 로직을 보여주는 인터랙티브 웹 데모입니다.

## 🛠 실행 방법

1. **프로젝트 폴더로 이동**
   ```bash
   cd utility-pro
   ```

2. **기본 구동 (이미 서버가 켜져 있을 수 있습니다)**
   ```bash
   npm run dev
   ```
   - 현재 서버는 **[http://localhost:5175](http://localhost:5175)** 에서 구동 중입니다.

## 📖 주요 데모 시나리오

1. **사용 제한 (Gating)**
   - 초기 접속 시 무료 이용권이 없습니다. 도구를 실행하려고 하면 광고 시청 모달이 뜹니다.
2. **광고 시청 (Ad Reward)**
   - 광고 시청 버튼 클릭 시 15초간 데모 광고가 재생되며, 완료 후 상단에 5분 이용권 타이머가 표시됩니다.
3. **프리미엄 구독 (Subscription)**
   - `Pricing` 페이지에서 프리미엄을 활성화하면 타이머가 무제한으로 바뀌며 광고 없이 모든 기능을 즉시 사용할 수 있습니다.
4. **가상 파일 처리**
   - 어떤 파일을 업로드해도 안전하게 분석 및 처리되는 과정을 시뮬레이션하며, 결과물을 실제로 다운로드해 볼 수 있습니다.

## 🏗 기술 스택

- TypeScript
- Tailwind CSS v4 (최신 기능 사용)
- Lucide React (아이콘)
